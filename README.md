This is a PaZcal compiler for the class Compilers (8th Semester NTUA)

For more info please refer to the docs. In order to build the documentation you
will need Sphinx.
To get Sphinx
```
# pip install sphinx
```

To build the documentation, in doc directory:
```
make html
```
For the impatient
```
make depend
make
./pazcal
```
